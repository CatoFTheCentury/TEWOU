# TEWOU Game Engine
TEWOU is a 161kb WebGL-powered 2D game engine meant to be usable for all types of game projects. Built entirely in Typescript, it is meant to be extended with a game, sideloaded on a webpage. API exposed to the window object, accessible through the `TEWOU` namespace.

TEWOU features animated sprites, hitbox collision detection and layered tile-based levels. Rendering that has a shader-ready pipeline, rotation, scaling and custom GLSL effects out of the box.

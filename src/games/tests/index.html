Test Structure

Each test is isolated and demonstrates specific engine functionality:

  • Animation: Frame-based animations and state switching
  • Physics: Movement, collision detection, and physics systems
  • Input: Keyboard handling and event registration
  • Time: Timeout and trigger systems
  • Composition: Frame and snap hierarchy
  • Integration: Complete game scenarios
